# Profile Image

Place your professional headshot here.

## Requirements:
- Format: JPG, PNG, or WebP
- Recommended size: 500x500 pixels or larger
- Aspect ratio: 1:1 (square)
- Quality: High resolution, good lighting

## Filename:
Rename your image to: `profile.jpg`

## Example:
```
profile/
  └── profile.jpg  ← Your photo goes here
```

The website will automatically load this image. If no image is found, a placeholder will be displayed.
